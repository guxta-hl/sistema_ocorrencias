# Sistema de Ocorrências e Apreensões

## 📌 Status do Projeto
Em desenvolvimento

## 💻 Tecnologias Utilizadas
- Java
- MySQL
- JDBC
- Swing

## 👨‍💻 Desenvolvedor
- Gustavo H. Lopes

## 🎯 Objetivo do Sistema
Sistema desenvolvido para cadastro e gerenciamento de ocorrências e apreensões, com foco na organização e controle de dados.

## ⚙️ Funcionalidades
- Cadastro de ocorrências
- Cadastro de apreensões vinculadas (1:1)
- Listagem de ocorrências
- Integração com banco de dados MySQL
- Relacionamento entre entidades com JOIN
